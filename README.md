# [bradic.dev](https://bradic.dev/) — Personal Engineering Portfolio

A minimalist personal website and digital showcase, designed to reflect a "system-level" engineering aesthetic. This platform serves as a central hub for my projects, technical experience, and professional identity.

---

## Tech Stack

- **Frontend:** [React](https://react.dev/) & [TypeScript](https://www.typescriptlang.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Motion:** [Framer Motion](https://www.framer.com/motion/)
- **Routing:** [React Router](https://reactrouter.com/) (HashRouter)

---

## Installation & Usage

To run the project locally for development or review:

1. **Clone the repository:**
  ```bash
  git clone https://github.com/Bruda04/personal-website.git
  cd personal-website
  ```
   
2. **Install dependencies:**
  ```bash
  npm install
  ```

3. **Launch the development server:**
  ```bash
  npm run dev
  ```
